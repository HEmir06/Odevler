# EG-KART 🎴

EG-KART, Unity motoru ile geliştirilmiş, Nesne Tabanlı Programlama (OOP) prensiplerini temel alan profesyonel bir kart oyunudur. Gelişmiş AI rakipler ve modern bir UI tasarımı sunar.

## ✨ Özellikler
- **Modern Neon Tasarım:** Canlı renkler ve premium kart görselleri.
- **Akıllı AI:** Zorlu rakiplere karşı stratejik oyun yapısı.
- **Kapsamlı OOP Mimarisi:** Inheritance, Polymorphism ve Abstraction prensipleriyle kodlanmış esnek yapı.
- **Güvenli Kurulum:** Windows tabanlı "Tek Seferlik Kurulum" kısıtlamalı profesyonel kurulum paketi.

## 🚀 Kurulum ve Çalıştırma

### Hazır Kurulum (EXE)
1. `Installer/` klasörüne gidin.
2. `EG-KART_Setup.exe` dosyasını indirin ve çalıştırın.
3. Kurulum tamamlandıktan sonra masaüstündeki kısayoldan oyuna başlayabilirsiniz.

### Kaynak Koddan Derleme (Unity)
1. Bu depoyu klonlayın.
2. Unity 2021.3+ sürümüyle projeyi açın.
3. `Assets/Scenes/GameScene` sahnesine gidin ve Build alın.

## 🛠 Kullanılan Teknolojiler
- **Motor:** Unity 2021.3.x
- **Dil:** C#
- **Kurulum:** Inno Setup 6+
